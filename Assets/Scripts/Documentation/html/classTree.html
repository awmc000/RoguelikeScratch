int [,] Tree.GenerateMap ( )
inline

The main feature of Tree. Generates an int array representing a dungeon generated by the binary partition algorithm, with all rooms connected by dog-leg hallways.

Returns
A 2-dimensional jagged int array where value of TileFlagFloor == 1 represents a floor tile and TileFlagWall == 0 represents a wall tile.
